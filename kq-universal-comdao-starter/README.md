# CommonDao

## 功能

- 简单舒适且难以维护的与数据库交互方式
- 你懂的

## 使用方式

引入依赖
```xml
<dependency>
    <groupId>com.github.lzj960515</groupId>
    <artifactId>kq-universal-comdao-starter</artifactId>
    <version>${revision}</version>
</dependency>
```
